# Space Radiation Monitoring System

A Java-based application that predicts radiation levels and recommends shielding using real-time data visualization.

## Features
- Radiation prediction model
- Dynamic shielding recommendation
- Real-time graph visualization
- Interactive UI (Java Swing)

## Technologies
- Java
- Swing (UI)
- Custom graph rendering

## How to Run
1. Compile:
   javac *.java
2. Run:
   java Main
